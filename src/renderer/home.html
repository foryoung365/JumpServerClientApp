JumpServer Community Windows Web Wrapper

把 JumpServer 社区版 Windows Web 会话装进桌面壳里。

此版本不改造 JumpServer 服务端，继续沿用原有登录、资产访问与会话进入流程，重点解决桌面客户端场景下的快捷键接管、中文输入与登录凭据保存体验。

连接配置

填写 JumpServer 地址后，主窗口会直接承载原站点；命中 Lion 会话链接时自动切换到会话增强模式。

登录凭据

尚未检测当前地址的登录凭据状态。

凭据仅针对 JumpServer Web 登录页生效，使用系统加密能力保存，不会写入普通配置文件。

自定义远端快捷键映射

把安全的本地触发键映射为远端组合键，适用于 Win+X、Win+R、Ctrl+Alt+Delete 等系统级动作。

格式示例：本地触发键填写 Ctrl+Alt+X，远端组合键填写 Win+X。

窗口增强

识别 Lion 会话路径后自动接入会话浮层，统一处理会话内快捷键、特殊按键与文本桥接。

快捷键接管

优先把 VS Code、编辑器与终端中的常用高频快捷键转发到远端，同时保留本地面板控制热键。

登录页体验

支持 JumpServer Web 登录页的账号密码自动回填，并在登录成功后询问是否保存或更新当前凭据。

默认本地热键

Ctrl+Alt+K 打开 Session 面板 Ctrl+Alt+Space 切换 Text Mode Ctrl+Alt+Enter 切换本地全屏
