JumpServer Community Windows Web Wrapper

把社区版 Lion 会话装进桌面壳里，用 Electron 先解决最痛的两个问题。

这版 MVP 不改 JumpServer 服务端，保留原登录与资产访问流程，重点接管 VS Code 高频快捷键，并提供“本机 IME 短句提交”的文本模式。

连接配置

填写 JumpServer 地址后，主窗口会直接承载原站点；命中 Lion 会话链接时自动分流到专用 Session 窗口。

自定义远端快捷键映射

把安全的本地触发键映射成远端组合键，适合 `Win+X`、`Win+R`、`Ctrl+Alt+Delete` 这类不适合物理直通的动作。

格式示例: 本地触发键填 `Ctrl+Alt+X`，远端组合键填 `Win+X`。当前只支持“一个主键 + 若干修饰键”的映射。

窗口分流

识别 `/connect/`、`/monitor/`、`/share/:id/` 这类 Lion 会话路径，新开无地址栏 Session 窗口。

快捷键接管

会话窗口拦截 `Ctrl+P`、`Ctrl+Shift+P`、`Ctrl+F`、`Ctrl+H`、`Ctrl+S`、`Ctrl+Tab`、`Alt+Left/Right`、`F11`。

双模输入

默认 Shortcut Mode 走远端按键；Text Mode 用本地 IME 完成中文，再通过“剪贴板 + 远端粘贴”提交。

默认本地热键

Ctrl+Alt+K 打开 Session 面板 Ctrl+Alt+Space 切换 Text Mode Ctrl+Alt+Enter 切换本地全屏
